# MERN Stack
MERN Stack 
----

## Technologies Used
- MongoDB
- Express.js
- React
- Node.js
- Material-UI

## Development
1. Start the application
### Start the server
```bash
cd server
npm install
npm run start
```

### Start the client
```bash
cd client
npm install
npm run start
```

2. Access the application
Open your web browser and go to http://localhost:3000 to use the application.
